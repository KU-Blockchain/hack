# Midwest Block-a-thon Placeholder

A minimal placeholder site for the next year's hackathon. Used while the full site is being rebuilt.

## Contents

- **StarField** – Animated starfield background
- **Logo** – The Midwest Block-a-thon logo
- **Finished message** – "The hackathon has finished."
- **Devpost button** – Links to view projects on Devpost

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
